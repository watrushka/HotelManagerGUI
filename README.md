# HotelManager

**HotelManager** — это приложение на C++ с графическим интерфейсом Windows, позволяющее управлять гостиницей:  
регистрация клиентов, управление комнатами, расчёт стоимости проживания, сортировка данных и работа с базой данных.

---

## Основные функции

- **Управление клиентами**:
  - Добавление, редактирование и удаление клиентов.
  - Учет количества дней проживания.
  - Расчёт общей стоимости проживания.

- **Управление комнатами**:
  - Добавление и удаление комнат.
  - Отслеживание занятости комнаты.
  - Тип комнаты и стоимость за день.

- **Сортировка**:
  - Сортировка комнат по стоимости за день, типу и ID.
  - Поддержка изменения направления сортировки при повторном нажатии.

- **Работа с базой данных**:
  - Сохранение и загрузка информации о комнатах и клиентах в SQLite.
  - Обновление базы при удалении или изменении данных в приложении.
  - Возможность синхронизации с текстовыми файлами через экспорт и импорт.

- **Файловый ввод/вывод**:
  - Сохранение и загрузка данных о комнатах и клиентах в текстовый файл.

---

## Используемые технологии

- C++ (Win32 API)
- SQLite для хранения данных
- ListView для отображения клиентов и комнат
- Диалоговые окна для ввода данных
- Стандартная библиотека C++

---

## Структура проекта

- **Hotel.h / Hotel.cpp** — управление комнатами и клиентами, синхронизация с БД.
- **Room.h / Room.cpp** — описание комнаты.
- **Client.h / Client.cpp** — описание клиента.
- **Database.h / Database.cpp** — работа с SQLite.
- **Main.cpp** — главный файл с GUI и обработкой событий.
- **Dialogs** — диалоговые окна для добавления/редактирования комнат и клиентов.

---

## Сборка и запуск

1. Установи SQLite и подключи библиотеку к проекту в Visual Studio.
2. Открой проект в Visual Studio.
3. Собери проект (Build Solution).
4. Запусти приложение (Start Debugging или Ctrl+F5).

---

## Пример работы

- Добавление комнаты → Добавление клиента → Клиент автоматически занимает выбранную комнату.  
- Сортировка комнат с возможностью смены направления при повторном нажатии.  
- Синхронизация данных с базой SQLite при сохранении и загрузке.  
- Редактирование и удаление клиентов и комнат с обновлением базы данных.
